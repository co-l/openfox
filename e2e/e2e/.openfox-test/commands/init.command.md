Read the entire codebase and generate or refresh the `AGENTS.md` file at the project root.

This file should serve as a comprehensive guide for AI coding agents operating in this repository. Include:

1. **Project Overview** - What the project does, tech stack, high-level architecture
2. **Directory Structure** - Key directories and what they contain
3. **Build, Lint, Test Commands** - How to build, run tests, lint, typecheck (both full suite and single file)
4. **Code Conventions** - Naming, file organization, patterns used in the codebase
5. **Key Abstractions** - Important classes, interfaces, and patterns an agent should understand
6. **Common Pitfalls** - Things that are easy to get wrong

Be thorough but concise. The target audience is an AI agent, not a human developer.
